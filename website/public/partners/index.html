i. the data you own

the between-visit record

The PMS captures 30 minute snapshots, fortnightly. Physitrack covers the 13 days in between. With API access, both land on the same screen.

Adherence in isolation is a number. Read alongside the clinical record, it's a signal: "60% completion in the two weeks following a discharge note flagging mild flare-up." Same data. Different page.
