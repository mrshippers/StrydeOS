## ElevenLabs plan

Current Ava setup uses ElevenLabs Conversational AI. Check your plan:
- **Starter** (free/low tier): Not enough minutes. Upgrade.
- **Scale** ($99/mo, ~11,000 min): Covers 1–3 clinics comfortably.
- **Business** ($330/mo): Only needed at 5+ clinics or heavy call volume.

For 3 Spires clinics (~6,600 min/mo estimated), **Scale is sufficient**.

## n8n note

The Railway deployment (`n8n-production-b7325.up.railway.app`) runs the **self-hosted community edition** of n8n — same software, just hosted on Railway containers instead of n8n's own cloud. This is the correct setup for production. n8n Cloud ($50+/mo) is unnecessary.
