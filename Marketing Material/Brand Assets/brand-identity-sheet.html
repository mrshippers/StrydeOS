01 — The Mark
The Monolith
A performance pillar. The diagonal face catches light — precision at the apex. Beneath it, three chevrons climb. Momentum contained, then released.
Container
Gradient glass — linear (#2E6BFF 58% → #091D3E 72%) + radial light source (#6AABFF 42%). Not a solid fill. rx:24 squircle.
Ghost Pillar
White at 7% opacity (rect 35,20 → 57,80, rx:5). Lower half darkened with black at 10%.
Catch-light
Diagonal cut from (35,52) → (57,40). Top face filled with white gradient (55%→97%). Edge stroke 55%.
3 Chevrons
Clipped inside pillar. Bottom (20%, sw:2) → Mid (42%, sw:2.5) → Top (72%, sw:3.2). Ascending.

05 — Usage Rules
Do / Don't
✓ DO
Use the gradient glass container for all standard applications
Use Solid Royal variant only for app icons, favicons, constrained contexts
Use On Cream colourway with royal blue chevrons on light backgrounds
Maintain minimum 16px — use simplified prop at ≤24px
Source all values from brand.ts — single source of truth
✗ DON'T
Never use a flat/solid fill for the standard mark — container is always gradient glass
Never rotate, skew, add external shadows, or alter the diagonal cut angle
Never place the mark on busy backgrounds without sufficient contrast
Never use old hex values: #1648BC #2968E0 — use #1C54F2, #2E6BFF
Never separate chevrons from the pillar — they are clipped inside, always
