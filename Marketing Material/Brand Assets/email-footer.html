05 — Installation
How to install in iCloud Mail
Three paths. Pick the one that matches how you send mail day-to-day.

A · iCloud Mail (web)

  1. Open this file in Safari (Chrome mangles the SVG gradients on paste).
  2. Click just before the blue top-bar of the signature you want, then shift-click just after strydeos.com to select the whole block.
  3. Cmd + C to copy.
  4. In iCloud Mail, hit the gear icon (top left) → Preferences → Composing.
  5. Tick Add a signature.
  6. Click the signature box and Cmd + V to paste.
  7. Done. Send yourself a test from iCloud Mail → check it renders in iOS Mail too.

B · Apple Mail (Mac)

  1. Mail → Settings → Signatures.
  2. Pick the @icloud.com account on the left.
  3. Click +, name it StrydeOS, type any placeholder.
  4. Uncheck Always match my default message font.
  5. Open this file in Safari, select and copy the signature block, return to Mail, paste over the placeholder.
  6. At the bottom, set Choose Signature → StrydeOS for that account.
  7. If the logo shows blank: close Mail, lock the file at ~/Library/Mail/V10/MailData/Signatures/*.mailsignature, reopen Mail.

C · iOS Mail (iPhone)

  1. Setting a signature per-account happens on the phone itself.
  2. On Mac, after setup A or B, sign in to iCloud Mail on iPhone via Safari.
  3. Long-press the rendered signature in a sent email, Select All, Copy.
  4. Settings → Mail → Signature → Per Account.
  5. Paste into the iCloud field. Shake to undo if it strips styling, then try again via the iOS Mail signature field directly.
  6. Plain-text fallback for iOS if styling keeps breaking: Jamal · Founder, StrydeOS · strydeos.com.
Two addresses, two signatures: Use jamal@strydeos.com with Option A for anything personal / outbound sales / founder replies. Use hello@strydeos.com with Option B for support, system, and general inbound. In iCloud Mail, set a different signature per address under Composing → Signature → pick the address from the dropdown above the text field.
Outlook caveat: If you ever expect recipients on Outlook desktop (Windows), swap the inline SVG logo for a hosted PNG <img> tag — Outlook's Word-based renderer ignores SVG. Not a problem for iCloud-to-iCloud, Gmail, Apple Mail, or any modern webmail.
